Create a React component called EcoScore.

Features:
- Displays environmental score out of 100
- Circular progress indicator
- Dynamic message based on score:
   80-100 → "Great! This item can be recycled"
   50-79 → "Moderate environmental impact"
   0-49 → "Consider better disposal options"
- Color system:
   Green = good
   Yellow = medium
   Red = harmful
- Styled using Tailwind CSS
